# Roadmap

Where Fund Ledger is headed. Updated manually or by Claude Code — not tied to
any particular release.

## Potential future work

- PDF output (pdf-lib): generate a printable PDF directly in addition to the
  HTML document.
- Offline-first PWA packaging so the app can be installed and launched without
  a browser address bar.
- Export/backup: zip the entire ledger folder to a single downloadable archive.
- Signature capture: draw or upload a signature image to embed in the document.
